# Appendix D: 3Gun-Specific Division Modifications

## D.1 Modified (MOD) Rifle
Any rifle allowed per [[Appendix B_Rifle Divisions#B.1.1 Competition (COMP)|Section B.1.1]] with the following adjustments:

### D.1.1 Maximum muzzle device size
1.0” diameter x 3.0” length measured from end of barrel crown to end of muzzle.

### D.1.2 Optical sighting system
Only a single optical sighting system is allowed.

### D.1.3 Bipod
A single bipod may be attached to the rifle, but must remain attached throughout the course of fire.

## D.2 Modified (MOD) Handgun
Any handgun allowed per [[Appendix A_Pistol Divisions#A.1.1 Competition (COMP)|Section A.1.1]] with the following adjustments:

### D.2.1 Max magazine length
141.25mm

### D.2.2 Minimum power factor
No

### D.2.3 Maximum rounds loaded in magazines at start signal
N/A

## D.3 Tactical Optics (TAC) Rifle
Any rifle allowed per [[Appendix B_Rifle Divisions#B.1.1 Competition (COMP)|Section B.1.1]] with the following adjustments:

### D.3.1 Optical sighting system
Only a single optical sighting system is allowed.

### D.3.2 Maximum muzzle device size
1.0” diameter x 3.0” length measured from end of barrel crown to end of muzzle.

## D.4 Tactical Optics (TAC) Handgun
Any handgun allowed per [[Appendix A_Pistol Divisions#A.1.1 Competition (COMP)|Section A.1.1]] with the following adjustments:

### D.4.1 Max magazine length
141.25mm

### D.4.2 Minimum power factor
No

### D.4.3 Maximum rounds loaded in magazines at start signal
N/A

### D.4.4 Sights
Iron Sights Only
