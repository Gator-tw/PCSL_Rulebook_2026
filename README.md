# PCSL General Rulebook - Markdown (2026)

This repository contains the 2026 PCSL (Practical Shooting Competition League) General Rulebook, converted from the official PDF into a structured Markdown format. 

## Objective
The sole purpose of this project is to convert the official rulebook into a high-quality text format that maximizes reading, parsing, and retrieval efficiency for AI models (LLMs) and RAG (Retrieval-Augmented Generation) systems.

## Disclaimer
This is a community-maintained repository optimized for digital parsing and AI development. It is not officially affiliated with the Practical Shooting Competition League. For official match rulings, arbitrations, and final authority, always refer to the original PDF published on **[PCSLeague.us](https://rules.pcsleague.com/)**.
