# Appendix C: Shotgun Divisions

## C.1 PCSL Shotgun - Core Divisions
There are no recognized divisions at this time.

## C.2 PCSL Shotgun - Recognized Divisions

### C.2.1 Open (OPEN)
| Criteria | Specification |
| :--- | :--- |
| **Minimum Caliber** | 20 Gauge |
| **Maximum Caliber** | 12 Gauge |
| **Minimum power factor** | no |
| **Shotgun requirement** | Must be a shotgun per [[02_Competitor Equipment#2.4.3 Shotgun|Section 2.4.3]] |
| **Other restrictions** | no |

### C.2.2 Modified (MOD)
| Criteria | Specification |
| :--- | :--- |
| **Minimum Caliber** | 20 Gauge |
| **Maximum Caliber** | 12 Gauge |
| **Minimum power factor** | no |
| **Magazine type** | Fixed single magazine tube only, no speed-loading devices |
| **Optical sight** | allowed - A single non-magnified optical sight is permitted |
| **Flashlights/lasers** | No restrictions on flashlights/lasers |
| **Barrel porting, Compensators, and Brakes** | allowed |
| **Maximum magazine tube capacity** | none |
| **Maximum rounds loaded at start signal** | 13 if starting chambered, 12 if starting with an empty chamber |
| **Support devices** | Bipods, bags, and other related support devices may not be used |
| **Shotgun requirement** | Must be a shotgun per [[02_Competitor Equipment#2.4.3 Shotgun|Section 2.4.3]] |
| **Other restrictions** | no |

### C.2.3 Practical (PRAC)
| Criteria | Specification |
| :--- | :--- |
| **Minimum Caliber** | 20 Gauge |
| **Maximum Caliber** | 12 Gauge |
| **Minimum power factor** | no |
| **Magazine type** | Fixed single magazine tube only, no speed-loading devices |
| **Optical sight** | allowed - A single non-magnified optical sight is permitted |
| **Flashlights/lasers** | No restrictions on flashlights/lasers |
| **Compensators, and Brakes** | no |
| **Barrel porting** | allowed |
| **Maximum barrel length** | 21.0in |
| **Maximum magazine tube capacity** | 8 |
| **Maximum rounds loaded at start signal** | 9 if starting chambered, 8 if starting with an empty chamber |
| **Support devices** | Bipods, bags, and other related support devices may not be used |
| **Shotgun requirement** | Must be a shotgun per [[02_Competitor Equipment#2.4.3 Shotgun|Section 2.4.3]] |
| **Other restrictions** | no |
