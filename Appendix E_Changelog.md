# Appendix E: Changelog

## E.1 Rule Changes and Updates
This page documents the history of changes to the PCSL General Rulebook.

### E.1.1 2026

#### E.1.1.1 Version 2026-05-05 Draftable Comparison
* Using new markdown-generated formatting
* .22Magnum added to [[Appendix A_Pistol Divisions#A.2.1 Rimfire Rifle (RFR)|A.2.1]] / [[Appendix A_Pistol Divisions#A.2.2 Rimfire Pistol (RFP)|A.2.2]] (Rimfire Rifle)
* Divisions
    * Removed from [[03_Match Types and Levels|Chapter 3]], added as Divisions [Appendix A_Pistol Divisions](Appendix%20A_Pistol%20Divisions.md), [[Appendix B_Rifle Divisions]], [[Appendix C_Shotgun Divisions]], and [[Appendix D_3Gun-Specific Division Modifications]]
    * Reformatted with tables for easier viewing
    * Practical Irons, TenRoundIrons, and Revo have been removed
* “Carbine” generally renamed to “Rifle” Except for PCCs
* Section number references fixed as needed

#### E.1.1.2 Version 2026-02-06 Draftable Comparison
* Rules referencing other sections’ numbers updated
* 2.10 Concealment added
* Chapter 3:
    * “Load-to capacity” terminology in section 3 changed
    * Actual Carry Pistol (ACP) Division removed
    * Defensive Practical (DP) and Defensive Competition (DC) Divisions added
    * Revolver Open (REVO) division renamed
    * 3.20.7 included 3-gun in match types allowing bipods and such
    * Muzzle Device section re-organized and rewritten. Brakes on the ends of suppressors are no longer permitted in Practical
* 4.9 Moved from Arbitration chapter
* 5.1.2.1 rolled into 5.1.2
* 5.1.4 added to restrict how much time is wasted during Make Ready trying out positions
* 7.7.2 Removed explicit reference to DQ sections. The rules are clear already
* 7.7.2.4 Added RM discretion on abandonment permissions
* 7.14 Added section. This was always implied previously, now it is clearly spelled out
* 9.11.3 Prone shooting areas must be fully defined all around now
* 9.14 New section making implied permission explicit.
* 11.4.2.1 Clarified rules confusion
* 11.4.3.3 / 11.4.3.4 Fixed requirement for Delegates to make themselves known. Anybody can be a Delegate
* 11.4.5.3 / 11.4.5.4 ROs should not be removing targets unless the RM allows it
* 11.6 Cleaned up wording and confusion around when a HIT call is required and when it is a courtesy
* 11.12 Moved from Arbitration chapter
* Chapter 16 ([[16_Arbitration|Chapter 16]]):
    * General rewrite
    * Moved Video evidence
    * Arb fee added
    * Arb committee, composition, and roles clarified
    * Arbitration Form added as a link
