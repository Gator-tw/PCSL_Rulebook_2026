# Appendix B: Rifle Divisions

## B.1 PCSL Rifle - Core Divisions

### B.1.1 Competition (COMP)
| Criteria | Specification |
| :--- | :--- |
| **PCC allowed** | Pistol Caliber Carbines as defined in [[Appendix A#A.1.3 Pistol Caliber Carbine (PCC)|Section A.1.3]] are allowed |
| **Minimum caliber** | 5.7x28, 5.45x39, 5.56x45, 9x19mm |
| **Maximum caliber** | .308 Win/7.62x51 |
| **Minimum power factor** | none for rifle calibers. When using PCC: 125 |
| **Optics** | no restrictions |
| **Suppressors** | allowed |
| **Support devices** | Bipods, bags, and related support devices are allowed in PCSL Rifle or 3-Gun matches in Competition division only, but not in PCSL 2-Gun matches. |
| **Tripods** | Tripods are never allowed outside of PCSL 3-Gun Open division |
| **Rifle requirement** | Must be a rifle per [[02_Competitor Equipment#2.4.2 Rifle|Section 2.4.2]] |
| **Other restrictions** | no |

### B.1.2 Practical (PRAC)
| Criteria | Specification |
| :--- | :--- |
| **PCC allowed** | no |
| **Minimum caliber** | 5.45x39, 5.56x45 |
| **Maximum caliber** | .308 Win/7.62x51 |
| **Minimum power factor** | no |
| **Optics** | no restrictions |
| **Maximum rounds loaded in magazines at start signal** | 30 |
| **High-capacity magazines** | Magazines that are capable of holding more than 30 rounds (including drums) are allowed provided that they are only loaded to 30 rounds at the start of the stage |
| **Coupled magazines** | Coupled magazines (side by side) are prohibited |
| **Magwells** | Magwells up to 2.000” outer width are allowed |
| **Suppressors** | allowed |
| **Muzzle brakes** | Muzzle brakes are prohibited UNLESS used for the purpose of adapting a suppressor, as long as the suppressor remains attached. |
| **Blast mitigation devices** | Blast mitigation device such as the Surefire Warden or Griffin Armament Blast Shield that remains attached to a muzzle brake is allowed as long as the device directs all gasses forward of the muzzle. |
| **Bolt Carrier Groups** | Lightened or Reduced Mass BCGs are prohibited. In any platform, the total assembled BCG mass must be no less than 0.2oz under the OEM BCG mass of the platform. In an AR-15, the OEM BCG mass is considered to be 11.0oz |
| **Support devices** | Tripods, bipods, bags, and related support devices are prohibited |
| **Rifle requirement** | Must be a rifle per [[02_Competitor Equipment#2.4.2 Rifle|Section 2.4.2]] |
| **Other restrictions** | no |

## B.2 PCSL Rifle - Recognized Divisions
There are no recognized divisions at this time.
